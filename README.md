SkillLink - Plataforma de Intercambio de Habilidades

SkillLink es una aplicación web desarrollada con Spring Boot que permite a estudiantes ofrecer y solicitar tutorías académicas dentro de una comunidad universitaria.

Tecnologías utilizadas

Java

Spring Boot

H2 Database (base de datos en memoria)

HTML / CSS (interfaz básica)

Funcionalidades principales

Registro de usuarios

Edición y eliminación de usuarios

Validaciones en formularios

Mensajes de confirmación para acciones (crear, editar, eliminar)

Base de datos

Se utilizó H2 Database para simplificar el desarrollo, evitando la necesidad de instalar un gestor externo como MySQL.

Notas importantes

Parte del conocimiento, estructura y lógica aplicada en este proyecto está basada en experiencia previa y código trabajado en un repositorio privado de carácter laboral, el cual no puede ser compartido públicamente.

Ejecución del proyecto

Para correr el proyecto:

./mvnw spring-boot:run

Luego abrir en el navegador:

http://localhost:8080
